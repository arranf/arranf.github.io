Blog Quandaries | Scriptogram, Postach.io, Ghost, and Github Pages

29 Nov 2014

My last three blog posts have been posted to three different web addresses using three different web services.

Choosing where to host my blog and which service to use has been a bit of an ordeal.

I initially started out using Scriptogram but their theme support didn't quite cut it. Postach.io was my next stop but they removed their Dropbox integration recently forcing me to relocate.

I now host my blog on Github Pages using a static version of a locally hosted Ghost blog (following this guide) saving myself £3/$5 a month over a dynamic implementation. Clunky but cheap.

All in all I've wasted an afternoon and evening trying to get it all working but I also played a little more thoroughly with Github, not quite a worthwhile trade but at least I've learnt a little from the process.
